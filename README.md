# AI Text Summarizer

Mini AI-powered automation tool built with Python and OpenAI API.

## Features
- Summarizes long text into short, clear summaries
- Uses OpenAI language models
- Simple command-line interface

## Technologies
- Python
- OpenAI API
- Prompt engineering

## How to run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt

3. Create a `.env` file and add your OpenAI API key: OPENAI_API_KEY=your_key_here

4. Run: python main.py


## Example
Input:


Artificial intelligence is transforming software development...


Output:


AI is changing how software is built by automating and improving processes.
